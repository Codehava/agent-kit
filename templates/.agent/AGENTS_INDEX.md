# Auto-Discovery Agent Persona Mechanism

> [!WARNING]
> STATIC INDEX OBSOLETE: We explicitly removed the hardcoded `AGENTS_INDEX.md` mapping file because maintaining an index for Vibe Coding personas is obsolete.

## Instructions for AI

You MUST NOT rely on an index file to discover Agent Personas. You are equipped with the **Auto-Discovery Engine**.

**Protocol:**
1. Determine the relevant keywords from the user request (e.g. `frontend-specialist`, `security-auditor`, `orchestrator`).
2. Run a keyword search on the directory using `find` or `grep_search` against `.agent/agents/` to adopt a character.
3. Notify the user indirectly that you have adopted the persona.
